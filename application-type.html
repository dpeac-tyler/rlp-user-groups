Application Type

To get started, please answer the following questions. The first set of questions will help you create the correct type of application.

  1. Does this application type need the individual/business to create an account?
    1a. If yes, will this application need an internal workflow review?
  2. Does this application need documents to be included as part of the submission?
  3. Is this application for an individual or a business?
    3a. Individual - Do you need to collect identification information for an individual?
    3a. Business - Do you need to collect identification information for business branches?
  4. Is there a fee for submitting this application?
  5. Does this application allow a license/permit?
    5a. If yes, does this application issue a certificate?
  6. Does the application require an electronic signature from the individual/business?

Note: For an Entity License, Individual, Business, and ID Collection options have been removed as they’re not entity related.

Please answer all questions.

IMPORTANT:   Responses to these questions are final.   Once the Application Type and Title Information are completed, you will not be able to edit these responses.
